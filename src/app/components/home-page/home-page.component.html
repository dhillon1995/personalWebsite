Development Kit

At first I made this dev-kit so I could easily test a can-bus project I was working on at the time. I wanted to make a basic OBD2 digital dash using a Raspberry Pi-3 with a can-bus hat and a touchscreen. I wanted to keep all the cables external as I wanted to be able to change components quickly, with that in mind I iterated through 3 designs. This is a great example of prototype development driven by real world testing. Since then I have used this devkit to test various automation and IOT projects.

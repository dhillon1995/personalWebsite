Garage Control Portal
The aim of the web-app side of this project was very simple, I needed a web portal that allows me to send a request to my webserver, which will then execute some code that opens the garage door without me having to carry a remote around. This project was a fairly simple one but it was great to implement these technologies to accomplish a real-world improvement. The web server is running on a raspberry Pi which is already in the garage, so attaching a relay board and some basic wiring to my existing garage door motor is all that was needed to set up the hardware environment. Another aspect to this project is that I used it as an excuse to test some basic TS/JS that I wrote in the routing module of angular. The code determines whether the browser is on a mobile or desktop, and depending on which it will serve completely different components. This may seem trivial in the world of responsive websites (like bootstrap), but this method allows you to have a completely different UI experience for mobile and desktop users. One does not hinder the other to retain cross platform compatibility like in bootstrap, instead you can have the UI optimized for large and small screens independently. This code can be seen working on the test page.

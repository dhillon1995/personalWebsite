Trip Planner Prototype
This project was a company project at my last place of employment. The goal was to leverage a business relationship we already had with a venue operator in the Cotswoldâ€™s. The website was to follow the general objective of TripAdvisor, so the home page consists of a rotating picture banner with a search bar above. You have a navigation bar that sticks to the top of the page to make exploring an easy experience, and a limited spaces remaining section showing venue cards. This basic venue card layout is carried over to the venue search pages. This website had a lot of promising features, such as an administrator portal which allows venue operators to conveniently view filter and edit individual websites. There was also a trip planner which would have allowed you to input specific data about your trip and what kind of things you like, so it could plan your trip for you. This prototype required quite a lot of TS/JS, JSON and MySQL alongside Python for the server side application.

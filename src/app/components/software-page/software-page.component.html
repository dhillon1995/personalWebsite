Garage Control Portal

This is a simple IOT/automation project I designed to open my garage door using any device able to browse the web, when connected to my Wi-Fi network. This means I don’t have to carry around my garage remote around when at home. It also allows me to open my garage door remotely for safe storage of delivery’s when I’m not at home. This project could easily mature into a more complete garage control system if I implement environment controls such as heating and lighting, or tool monitoring by integrating the control system from my 3-d printer, alongside a live webcam.

This Portfolio

This portfolio website was put together using technology that I’m very familiar with and have used in production environments. It’s primarily built using the Angular Framework created by google, which brings a lot of useful and convenient tools and capabilities along, such as the powerful CLI and component-based architecture. The languages I use are HTML 5, CSS 3, TypeScript & JavaScript (ES6), as well as supporting markups like JSON. This website is a great demonstration of my front-end capabilities, where as I usually focus on more back-end/functional or embedded programming.

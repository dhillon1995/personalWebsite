My First Website
The goal for this portfolio website is to show some of my skills using professional and personal projects that require different languages and technologies to undertake. The website is fairly simple in layout, it has 3 main pages; Home, Software Repo and Hardware Repo. The Home page showcases icon-based navigation with some custom CSS to implement hover and click animations. The hardware and software buttons take you to their respective pages, and the GitHub button takes you to my GitHub home page. The Hardware and software pages are also very simple in layout, this is by design so the user can easily see all of my projects without having read into them with a lot of detail, unless they wish to, in which case each project has an in detail click through.
